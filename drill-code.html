드릴 코드 생성

자동 채번

edit_note 입력 항목

엑셀에서 품번 + 드릴 최종 (+ BR 팁 + 단번호) 셀들을 한꺼번에 복사해서 여기에 붙여넣으면 자동으로 분배됩니다.

10자리 코드를 입력하세요. 이 코드 기준으로 하위 채번이 진행됩니다.

채번 규칙 요약

  • 드릴 초도: 완제품 코드의 1번째→2, 10번째→1 (1로 시작하는 경우)
  • 드릴 홈: BR 팁이 있거나 품번이 "A"로 끝날 때만, 1→2, 10→4
  • 드릴 단: A 끝 또는 BR 팁이 있을 때, 품번 마지막 영문(P/C/a/M)에 따라 직경 ±5
  • 센터/자재: 품번이 E로 시작/3번째 E면 "소재확인", BR 팁 끝이 B~F면 "0D3175*018", A 끝+BR 없음이면 "2R3175A1Z7"
