자동 채번

edit_note 입력 항목

엑셀에서 품명 + 품번 (+ BR 팁 + 단번호) 셀들을 한꺼번에 복사해서 어느 칸에든 붙여넣으면 자동으로 분배됩니다.

채번 규칙 요약

  • 드릴 초도: 완제품 코드의 1번째→2, 10번째→1 (1로 시작하는 경우)
  • 드릴 홈: BR 팁이 있거나 품번이 "A"로 끝날 때만, 1→2, 10→4
  • 드릴 단: A 끝 또는 BR 팁이 있을 때 — 기본은 품번 마지막 영문(P/C/a/M)에 따라 직경 ±5, BR이 L1/L2/L3로 끝나면 직경 그대로 + N자리에 L1→G, L2→J, L3→K
  • 센터/자재: 품명이 E로 시작/3번째 E면 "소재확인", BR 팁 코드(M1~M5, L1B~L3P, A1B~A5L 등)는 매핑 테이블에서 룩업, 없으면 단일 B~F 폴백, A 끝+BR 없음이면 "2R3175A1Z7"
